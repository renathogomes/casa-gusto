# Casa Gusto — versão com detalhes dos produtos

Site em Astro baseado na versão original do GitHub, mantendo o visual da home e usando fotos reais apenas na área dos produtos.

## O que foi atualizado

- 23 produtos cadastrados.
- Fotos reais mantidas somente no catálogo e nos detalhes dos produtos.
- Ao clicar em qualquer produto, abre uma janela com:
  - descrição completa;
  - diferenciais do produto;
  - harmonizações sugeridas;
  - sugestão de apresentação;
  - produtos relacionados;
  - botão de pedido pelo WhatsApp.
- Busca e filtros por categoria mantidos.
- Build gerado em `dist`.

## Rodar localmente

```bash
npm install
npm run dev
```

No Windows PowerShell, caso o `npm` esteja bloqueado, use:

```powershell
npm.cmd install
npm.cmd run dev
```
